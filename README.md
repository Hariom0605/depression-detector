🧠 Early Depression Detection using Facial Micro-expressions

A real-time depression screening tool using deep learning and facial emotion analysis.

🚀 Features
- Real-time facial emotion detection using DeepFace CNN model
- PHQ-9 inspired depression risk scoring
- Live emotion history graph (Chart.js)
- Mental health resources (India)

🛠️ Tech Stack
- Python, Flask, OpenCV
- DeepFace (FER2013 trained CNN)
- Chart.js, HTML, CSS

▶️ How to Run
pip install -r requirement.txt
python app.py

Then open: http://localhost:5000
